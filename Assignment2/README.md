# Personal Website – Antonia Schmid

## Overview

The website presents my academic background and interests, combining my studies in Archaeology and Creative Computing.

It consists of three main pages:
- **Welcome (Summary)** 
- **Archaeology** 
- **Creative Computing** 
## Features

- Multiple HTML pages connected via navigation
- Use of core HTML elements:
  - Headings
  - Paragraphs
  - Lists
  - Links
- Clean and minimal CSS styling
- Simple JavaScript interaction (button with alert)

## Technologies Used

- HTML
- CSS
- JavaScript

## JavaScript Functionality (Bonus)

A button on the homepage triggers a small dialog using JavaScript.

### What worked
- The button correctly triggers a function via `onclick`
- The alert is displayed in the browser
- Proper linking of the script file enables functionality

### What didn’t work initially
styling took me a while as always but there was nothing that didnt work for me 
## How to Run

1. Download all files
2. Place them in the same folder:
   - `index.html`
   - `archaeology.html`
   - `creative.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in a browser
